# Visual Studio for Mac — Keyboard Shortcuts

## Build & Run

| Action | Shortcut |
|--------|----------|
| Build All | ⌘B / F6 |
| Build | ⌘K / ⇧F6 |
| Start Debugging | F5 |
| Start Without Debugging | ⌥⌘⏎ / ⌃F5 |
| Stop Debugging | ⇧⌘⏎ / ⇧F5 |
| Step Into | ⇧⌘I / F11 |
| Step Over | ⇧⌘O / F10 |
| Step Out | ⇧⌘U / ⇧F11 |
| Run To Cursor | ⌘F10 |

## Navigation

| Action | Shortcut |
|--------|----------|
| Go to Declaration | ⌘D / F12 |
| Go to Implementation | ⌘I |
| Find References | ⇧⌘R / F12 |
| Go to File | ⇧⌘F |
| Go to Line | ⌘L |

## Editing

| Action | Shortcut |
|--------|----------|
| Format Document | ⌃I |
| Toggle Line Comment | ⌘/ |
| Duplicate Line | ⇧⌘D |
| Delete Line | ⌘⌫ |
| Move Line Up/Down | ⌥↑ / ⌥↓ |
| Indent / Unindent | ⌘] / ⌘[ |
| Rename | ⌘R / F2 |
| Show Code Generation | ⌘I |

## File

| Action | Shortcut |
|--------|----------|
| New File | ⌘N |
| New Solution | ⇧⌘N |
| Open | ⌘O |
| Save | ⌘S |
| Save All | ⇧⌘S |
| Close File | ⌘W |
| Close All | ⌘W (repeated) |

## Search

| Action | Shortcut |
|--------|----------|
| Find in File | ⌘F |
| Find / Replace | ⌥⌘F |
| Find in Files | ⇧⌘F |

## Debug

| Action | Shortcut |
|--------|----------|
| Toggle Breakpoint | ⌘\ / F9 |
| Enable/Disable Breakpoint | ⌥⌘/ / ⌘F9 |
| View Breakpoints | ⌥⌘B |
| Remove All Breakpoints | ⇧⌘F9 |
